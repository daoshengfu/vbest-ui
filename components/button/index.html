# button按钮

基础组件，用户交互、触发业务时使用

使用方法： 你只需要在任何页面使用<b-button>内容</b-button>即可生成一个简单的按钮，也可通过追加属性来增加样式。

# 基础按钮

#

#

代码示例：

      <b-button type="default">default</b-button>
      <b-button type="primary">primary</b-button>
      <b-button type="dashed">dashed</b-button>
      <b-button type="text">text</b-button>
      <b-button type="info">info</b-button>
      <b-button type="success">success</b-button>
      <b-button type="warning">warning</b-button>
      <b-button type="error">error</b-button>

通过type来指定按钮的类型。

# 圆角按钮

#

代码示例：

      <b-button type="info" round>info</b-button>
      <b-button type="success" round>success</b-button>

通过round属性将一个按钮设置成圆角按钮。

# 幽灵按钮:背景透明，可用于有背景色的区域。

代码示例：

      <b-button type="info" round transparant>info</b-button>
      <b-button type="success" round transparant>success</b-button>

通过transparent属性将一个按钮设置成幽灵按钮。

# 图标按钮

建议使用iconfont字体图标,使用svg需要自己微调样式。可通过width、height和circle属性组合设计自己喜欢样式的按钮。

代码示例：

    <b-button type="success" circle class="iconfont icon-success"/>
    <b-button type="warning" circle class="iconfont icon-success"/>

# 按钮尺寸

在VbestUI中button可通过size属性，或者width、height属性设置按钮大小。

# size

size：类型String 值
long，width:100%
large，width为300px
default，width为150px
small，width为80px

# 自定义按钮宽高：width,height

<b-button type="primary" width="60px" height="30px">primary</b-button>

# button API

属性 说明 类型 默认值
type 用于指定按钮的类型,可选值为 default、primary、joker、dashed、text、info、success、warning、error或者不设置 String default
size 用于指定按钮的尺寸,可选值为 long、large、default、small或者不设置 String default
round 设置按钮的圆角,可不设置 Boolean false
circle 用于设置按钮是否为圆形按钮 Boolean false
transparent 设置按钮背景是否镂空 Boolean false
disabled 是否禁用按钮 Boolean false
font-size 用于自定义按钮字体大小 String 浏览器默认字体大小
width 用于自定义按钮的width,可不设置 String 150px
height 用于自定义按钮的height,可不设置 String 40px
color 用于自定义按钮文字颜色,可不设置 String none
background 用于自定义按钮的背景颜色,可不设置 String none
border-color 用于自定义按钮边框颜色,可不设置 String none

# 事件回调

事件 说明 参数
