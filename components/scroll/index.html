scroll内部会监视滚动，并在合适的时机发射上拉加载更多事件pullingUp，使用时只需要监听pullingUp事件，并做一定的处理即可实现无线滚动。

点击查看代码 
<template>
    <div class="three">
      <b-scroll height="300px" @pullingUp="handlePullingUp">
        <ul>
          <li v-for="(item,index) in count2" :key="index">{{item}}</li>
        </ul>
      </b-scroll>
    </div>
</template>
<script>
export default {
  data() {
    return {
      count2:40
    };
  },
  methods:{
      handlePullingUp(){
          this.count2+=10;
      }
  }
};
</script>

li{

  margin: 0px 0px 10px 0px;

}
</style>

是否禁用滚动:
禁用
开启
手动设置滚动位置:

属性is-disable设置是否禁用滚动条，使用scroll提供的方法scrollTo可手动滚动到指定位置，具体参数请参考下面API。

点击查看代码 
<template>
<div class="four">
      <b-scroll height="300px" :is-disable="isDisable" ref="scroll">
        <ul>
          <li v-for="(item,index) in count" :key="index">{{item}}</li>
        </ul>
      </b-scroll>
      <div class="contr">
        <div class="item">
          <span class="title">是否禁用滚动:</span>
          <b-switch class="item" v-model="isDisable" size="large" true-color="var(--error)">
            <span slot="open">禁用</span>
            <span slot="close">开启</span>
          </b-switch>
        </div>
        <div class="item">
          <span class="title">手动设置滚动位置:</span>
          <b-button type="primary" @click="handleClick">滚动</b-button>
        </div>
      </div>
    </div>
</template>
<script>
export default {
 data() {
    return {
      isDisable: false,
      count: 40,
      scrollY: 50,
    };
  },
  methods: {
    handleClick() {
      this.$refs.scroll.scrollTo(-this.scrollY, 0.5);
      this.scrollY += 40;
    },
  },
};
</script>
<style scoped>
li{
  border: 1px solid var(--border);
  margin: 0px 0px 10px 0px;
  text-align: center;
  font-size: 18px;
}
</style>

# API

# Scroll属性

属性 说明 类型 默认值
height 设置Scroll的高度，也可通过css样式设置scroll高度 String 无
theme 设置Scroll滚动条主题，可选值：light、dark String light
speed 鼠标滚动滚动速度，默认30px Number 30
is-disable 是否禁用所有滚动 Boolean false
disable-wheel
version 0.2.2
 禁用mousewheel滚动 String 无
disable-bar 是否禁用滚动条，禁用时仍可通过滚轮滚动 Boolean false
pulling-delay
version 0.2.2
 滚动时要触发上拉加载更多、下拉刷新事件的延迟时间，小于这个时间不触发 Number 200ms
pulling-distance
version 0.2.2
 上拉加载更多、下拉刷新事件的触发距离，可滚动距离小于时触发 Number 50
round 设置滚动条为圆角样式 Boolean false
bar-color 设置滚动条背景颜色 String 无
bar-active-color 设置滚动条活跃时颜色 String 无
load-color 设置滚动条小球颜色 String 无
load-active-color 设置滚动条小球活跃时颜色 String 无

# Scroll事件回调

事件 说明 参数
scroll 滚动时触发，包括滑轮和鼠标拖动 Y轴滚动的位置
wheel
version 0.2.2
 鼠标滑轮滚动时触发 Y轴滚动的位置
moveStart
version 0.2.2
 鼠标按下滚动条小球时触发 Y轴滚动的位置
moving
version 0.2.2
 鼠标拖动滚动条时触发 Y轴滚动的位置
moveEnd
version 0.2.2
 结束拖动滚动条滚动时触发 Y轴滚动的位置
pullingDown 下拉刷新 无
pullingUp 上拉加载更多 无

# Scroll方法

方法 说明 参数
scrollTo 用于设置滚动到指定位置，接收三个参数. y:Y轴滚动位置，单位px。duration：滚动时间，单位s、类型Number。judge是否进行区间判断，默认为true。
refresh 当scroll新增大量内容时用于刷新滚动条，可使滚动更精确 无
