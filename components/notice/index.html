# Notice通知、提醒

一般用于在页面右上角显示全局通知、提醒。

# 基础用法   
version 0.1.8：鼠标移入不会关闭通知

使用this.$Notice.open(option)打开一个通知提醒，option是一个对象，里面有icon、type、duration、title、desc、等几个属性，分别用于设置通知是否使用图标，通知类型、通知时间、通知标题、内容。默认通知时间为4s。

点击查看代码 
<template>
  <div class="notice-page">
    <div class="box">
      <b-button type="primary" @click="open1">默认通知</b-button>
    </div>
  </div>
</template>
<script>
export default {
  methods: {
    open1() {
      this.$Notice.open({
        title: "我是默认信息标题",
        desc:
          "我是默认内容，我是默认内容，我是默认内容，我是默认内容，我是默认内容，我是默认内容，",
      });
    },
  },
};

# 带状态信息的Notice

除了open方法,VbestUI中Notice还提供了info、success、warning、error方法，它们是open方法的简化版，使用它们你不需要设置icon、type属性。

点击查看代码 
<template>
  <div class="box">
    <div class="tow item">
      <b-button type="info" @click="infoClick">信息通知</b-button>
      <b-button type="success" @click="successClick">成功通知</b-button>
      <b-button type="warning" @click="warningClick">警告通知</b-button>
      <b-button type="error" @click="errorClick">失败通知</b-button>
    </div>
  </div>
</template>
<script>
export default {
  methods: {
    infoClick() {
      this.$Notice.info({
        title: "info标题",
        desc: "...省略",
      });
    },
    successClick() {
      this.$Notice.success({
        title: "success标题",
        desc: "...省略",
      });
    },
    warningClick() {
      this.$Notice.warning({
        title: "warning标题",
        desc: "...省略",
      });
    },
    errorClick() {
      this.$Notice.error({
        title: "error标题",
        desc: "...省略",
      });
    },
  },
};
</script>

# 自定义Notice时长

duration用于设置一个Notice的显示时间，Number或String类型

点击查看代码 
<template>
  <div class="box">
    <div class="four item" v-if="line=='4'">
      <b-button type="info" @click="openTime">打开一个10sNotice</b-button>
    </div>
  </div>
</template>
<script>
export default {
  methods: {
    openTime(){
        this.$Notice.open({
            duration:10,
            title:'这是一个10s通知',
            desc:'这是一个10s通知,这是一个10s通知,这是一个10s通知,这是一个10s通知,这是一个10s通知,这是一个10s通知,这是一个10s通知'
        })
    }
  },
};
</script>

# Notice API

# 全局使用 Notice instance


  • this.$Notice.open(option)

  • this.$Notice.info(option)

  • this.$Notice.success(option)

  • this.$Notice.warning(option)

  • this.$Notice.error(option)

以上方法用于全局使用Notice，option为对象，具体说明如下:

属性 说明 类型 默认值
icon 是否使用图标 Boolean false
duration 设置Notice显示的时长，默认4秒，可不设置 Number或String 4
type 用于设置Notice的类型，可选值有：info,success,warning,error String 无
title Nitice标题 String 无
desc Notice内容 String 无

注： info、success、warning、error,可不设置icon和type。
