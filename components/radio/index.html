# radio单选框

基本表单组件，常用于多选一。为了与原生区分VbestUI中单选框写法为<b-radio></b-radio>

# 单独使用

# 组合使用

男

组合使用时，属性label用于设置radio的值，v-model用于绑定数据。

代码示例：

点击查看代码 
<template>
    <div>
      <b-radio label="男" v-model="value1">男</b-radio>
      <b-radio label="女" v-model="value1">女</b-radio>
      <div>{{value1}}</div>
    </div>
</template>
<script>
export default {
  data() {
    return {
      value1: "男",
    };
  },
};
</script>

# 实际使用中更建议搭配radio容器，它会让你更加方便

JavaScript

它的优点在于不用一个一个radio的写v-model，你只需在容器上绑定数据即可。

点击查看代码 
<b-radio-group v-model="value2">
    <b-radio label="java">java</b-radio>
    <b-radio label="JavaScript">JavaScript</b-radio>
    <b-radio label="python">python</b-radio>
    <div class="show">{{value2}}</div>
</b-radio-group>

# 显示边框

篮球

你只需要给radio容器加上属性border即可生成带边框的单选框。也可单给radio设置border属性

点击查看代码 
<b-radio-group v-model="value3" border>
    <b-radio label="篮球">
        <div class="iconfont icon-icon10">篮球</div>
    </b-radio>
    <b-radio label="足球">
        <div class="iconfont icon-swticonzuqiu">足球</div>
    </b-radio>
    <b-radio label="乒乓球">
        <div class="iconfont icon-pingpangqiu">乒乓球</div>
    </b-radio>
    <div class="show">{{value3}}</div>
</b-radio-group>

# 按钮样式的单选框

龙

属性button用于设置radio容器内的单选框为按钮类型。也可单给radio设置button属性

点击查看代码 
<b-radio-group v-model="value4" button>
    <b-radio label="猫">
        <div class="iconfont icon-buoumaotubiao45">猫</div>
    </b-radio>
    <b-radio label="狗">
        <div class="iconfont icon-gouxiao1">狗</div>
    </b-radio>
    <b-radio label="龙">
        <div class="iconfont icon-longxiao">龙</div>
    </b-radio>
    <div class="show">{{value4}}</div>
</b-radio-group>

# 垂直排列的单选框

龙

给radio添加属性vertical可实现容器内单选框垂直排列。

点击查看代码 
<b-radio-group v-model="value5" vertical>
    <b-radio label="猫">
        <div class="iconfont icon-buoumaotubiao45">猫</div>
    </b-radio>
    <b-radio label="狗">
        <div class="iconfont icon-gouxiao1">狗</div>
    </b-radio>
    <b-radio label="龙">
        <div class="iconfont icon-longxiao">龙</div>
    </b-radio>
    <div class="show">{{value5}}</div>
</b-radio-group>

# 禁用单选框

足球

给单个radio添加属性disabled可设置该单选框禁用

<b-radio label="足球" disabled>
    <div class="iconfont icon-swticonzuqiu">足球</div>
</b-radio>

# Radio API

# Radio属性

属性 说明 类型 默认值
value 不建议设置，建议用v-model替代 Number，String，Boolean 无
label 单选框的值 Number，String，Boolean 无
name 单选框的name String 无
border 用于设置单选框是否显示边框 Boolean false
button 用于设置单选框是否为按钮样式，优先级比border高 Boolean false
disabled 用于设置单选框是否禁用 Boolean false

# Radio事件

事件名 说明 参数
change 用于监听选项状态的改变 无

# Radio插槽

插槽名 说明
无 单选框的内容

# RadioGroup API

# RadioGroup属性

属性 说明 类型 默认值
value 不建议设置，建议用v-model替代 Number，String，Boolean 无
border 用于设置容器内部的单选框显示边框 Boolean false
button 用于设置容器内部单选框是否为按钮样式,优先级高于border false
vertical 用于设置容器内部单选框是否为垂直排列 Boolean false

# RadioGroup事件

事件名 说明 参数
change 用于监听选项状态的改变 无
