# Table 表格

常用于大量数据的展示

# 使用

点击查看代码 
<template>
    <div class="one">
      <b-table show-border stripe stripe-background-color="#f9f9f9" height="400px" class="table">
        <b-table-head slot="head" split split-color="var(--border)">
          <b-table-tr>
            <b-table-td width="50px"></b-table-td>
            <b-table-td width="60px">操作</b-table-td>
            <b-table-td>音乐标题</b-table-td>
            <b-table-td>歌手</b-table-td>
            <b-table-td>专辑</b-table-td>
            <b-table-td width="120px">时长</b-table-td>
          </b-table-tr>
        </b-table-head>
        <b-table-body slot="body">
          <b-table-tr v-for="(item, index) in count" :key="index">
            <b-table-td width="50px">
              {{ getListIndex(index) }}
            </b-table-td>
            <b-table-td width="60px"
              ><i class="iconfont icon-xihuan"></i
            ></b-table-td>
            <b-table-td>{{ song.name }}</b-table-td>
            <b-table-td>{{ song.artist }}</b-table-td>
            <b-table-td>{{ song.album }}</b-table-td>
            <b-table-td song="120px">{{ song.time }}</b-table-td>
          </b-table-tr>
        </b-table-body>
      </b-table>
    </div>
</template>
<script>
export default {
  data() {
    return {
      count: 40,
      song: {
        name: "青花瓷",
        artist: "周杰伦",
        album: "我很忙",
        time: "03:59",
      },
    };
  },
  methods: {
    /**获取音乐列表下标 */
    getListIndex(index) {
      let currentIndex = 0;
      currentIndex = index < 9 ? "0" + (index + 1) : index + 1;
      return currentIndex;
    },
  },
};
</script>

# Table API

# Table属性

属性 说明 类型 默认值
height 设置Table的高度、超出高度使用虚拟滚动条 String 无
stripe 开启斑马线 Boolean false无
stripe-background-color 斑马线背景颜色 String 无
split 是否显示列表分割线 Boolean false
split-color 分割线颜色 String 无
show-border 是否显示table边框 Boolean false
border-color table边框颜色 String 无
warp table中td文字是否换行 Boolean false

# Table插槽

插槽名 说明
head table表头
body table表体
