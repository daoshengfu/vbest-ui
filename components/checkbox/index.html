# checkbox复选框

基本组件，常用于多项选择，或单独状态切换。VbestUI复选框标签为<b-checkbox></b-checkbox>

# 单独使用

false

单独使用可用v-model绑定一个Boolean类型的值。

点击查看代码 
<template>
  <div class="box">
      <b-checkbox  v-model="value">喜欢？</b-checkbox>
      <div class="show">{{value}}</div>
  </div>
</template>
<script>
export default {
  data() {
    return {
      value: false,
    };
  },
};
</script>

# 组合使用

[ "龙" ]

属性label用于设置选项的值，给复选框容器设置v-model

值得注意的是： 当你组合使用时，容器v-model绑定的数据必须是Array类型

点击查看代码 
<template>
  <div>
    <b-checkbox-group v-model="value1">
      <b-checkbox label="猫">
          <div class="iconfont icon-buoumaotubiao45">猫</div>
      </b-checkbox>
      <b-checkbox label="狗">
          <div class="iconfont icon-gouxiao1">狗</div>
      </b-checkbox>
      <b-checkbox label="龙">
          <div class="iconfont icon-longxiao">龙</div>
      </b-checkbox>
    </b-checkbox-group>
    <div class="show">{{value1}}</div>
 </div>
</template>
<script>
export default {
  data() {
    return {
      value1: ["龙"],
    };
  },
};
</script>

# Checkbox API

# Checkbox属性

属性 说明 类型 默认值
value 只有单独使用时有效，可以使用v-model绑定数据 Boolean false
label 组合使用时设置单选框的值 Number，String，Boolean 无
name 复选框的name String 无
border 用于设置复选框是否显示边框 Boolean false
button 用于设置复选框是否为按钮样式，优先级比border高 Boolean false
disabled 用于设置复选框是否禁用 Boolean false

# Checkbox事件

事件名 说明 参数
change 用于监听复选框状态的改变，只有单独使用时有效，返回当前复选框的状态 更新后的值

# Checkbox插槽

插槽名 说明
无 复选框的内容，默认为label值

# CheckboxGroup API

# CheckboxGroup属性

属性 说明 类型 默认值
value 不建议设置，用v-model绑定数据 Array []
border 用于设置容器内部的复选框显示边框 Boolean false
button 用于设置容器内部复选框是否为按钮样式,优先级高于border false
vertical 用于设置容器内部复选框是否为垂直排列 Boolean false
disabled 用于设置容器内所有复选框是否禁用 Boolean false

# CheckboxGroup事件

事件名 说明 参数
change 用于监听容器内复选框选项状态的改变 Array
