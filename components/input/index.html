# input输入框

基本表单组件，VbestUI对原生input进行了功能丰富的扩充，让你更省心。

注： 为了与原生input区分，VbestUI中input组件写法为<b-input></b-input>

# 默认Input

用法： 你只需要，在任何你需要输入框的地方，输入<b-input/>即可生成一个简单的输入框，round属性是否使用圆角输入框，可用v-model绑定数据。

点击查看代码 
<template>
    <div class="group">
        <b-input v-model="value"/>
    </div>
</template>
<script>
export default {
    data(){
        return {
            value:''
        }
    }
}
</script>

# 带清除按钮的input   
version 0.1.8：文本为空时不显示清空按钮

属性clearable用于设置是否显示清空按钮。

点击查看代码 
<template>
    <div class="group">
        <b-input v-model="value" clearable/>
    </div>
</template>
<script>
export default {
    data(){
        return {
            value: "Hello World",
        }
    }
}
</script>

# 密码框

普通密码框


带查看密码按钮的密码框   
version 0.1.8：查看密码时输入密码，密码不可见

属性show-password用于设置显示查看密码按钮，查看密码时输入密码，为了安全性会自动密码不可见。

点击查看代码 
<template>
    <div class="group">
        <b-input type="password" v-model="value2"></b-input>
        <b-input type="password" v-model="value3" show-password></b-input>
    </div>
</template>
<script>
export default {
    data(){
        return {
            value2: "123456",
            value3: "123456",
        }
    }
}
</script>

自定义input图标


自定义前置图标

自定义后置图标

自定义前后置图标

VbestUI中input自定义图标非常简单、强大，你只需要给属性prefix、suffix传入适合的iconfont字体图标即可。

注: 值得一提的是当自定义后置图标时，内置的clearable、search、show-password属性将失效。

示例: <b-input suffix="iconfont icon-touxiang" v-model="value5" clearable/>

点击查看密码 
<b-input prefix="iconfont icon-touxiang" v-model="value4"/>
<b-input suffix="iconfont icon-touxiang" v-model="value5"/>
<b-input prefix="iconfont icon-touxiang" 
         suffix="iconfont icon-sousuo" 
         suffix-icon-size="22px"
         placeholder="请输入要搜索的人名" 
         v-model="value11"/>

# 搜索框

默认搜索框

默认搜索框的使用方式十分简单，你只需要添加一个属性 search即可生成一个搜索框。

代码示例：

<b-input search v-model="value6"/>

# 组合搜索框

https://

你可以根据VbestUI中input提供的插槽，随心所欲的按照自己的喜好设计一个功能强大的复合型输入框。

# 代码示例：

    <b-input v-model="value7">
        <div class="head" slot="head">https://</div>
        <div class="footer" slot="footer">
            <b-button type="primary" size="small" height="32px">确定</b-button>
        </div>
    </b-input>

# 其他input

只读输入框


禁用输入框


有长度限制的输入框   
version 0.1.8：达到最大长度时提供回调


添加属性disabled、readonly、maxlength可让输入框有对应的功能。事件inhibiting在输入字符达到最大限制长度时触发。

点击查看代码 
<template>
  <div>
      <h4>只读输入框</h4>
      <b-input v-model="value12" readonly/><br>
      <h4>禁用输入框</h4>
      <b-input v-model="value13" disabled /><br>
      <h4>有长度限制的输入框</h4>
      <b-input v-model="value14" placeholder="最多输入10个字符" :maxlength="10" @inhibiting="handleInhibiting"/>
  </div>
</template>
<script>
export default {
    data(){
        return {
            value12: "Hello World",
            value13: "123456",
            value14: "",
        }
    },
    methods:{
      handleInhibiting(value,size){
            this.$Toast.info(`${'达到长度限制,value:'+value+',限制长度：'+size}`,4);
        }
  }
}
</script>

# API

# Input属性

属性 说明 类型 默认值
value/v-model 用于设置Input的值，可用v-model替代。 String 无
type 用于设置Input的类型，可选值有：text、password、url、email、date、number、tel String text
size 用于设置Input的大小，可选值有：large、default、small String default
round 用于设置Input是否是圆角 Boolean false
search 用于设置Input是搜索框，带有搜索后缀图标。当type为password或者有clearable、show-password属性时不起效，存在属性suffix时无效。 Boolean false
clearable 是否显示清空按钮，存在属性suffix时无效。 Boolean false
show-password 是否显示查看密码按钮，只有当type值为password时生效，存在属性suffix时无效。 Boolean 默认值
prefix 用于自定义Input头部图标，可是任意的iconfont。 String 无
suffix 用于自定义Input尾部图标，可是任意的iconfont。 String 默认值
prefix-icon-size 自定义头部图标大小 String 无
suffix-icon-size 自定义尾部图标大小 String 无
disabled 设置Input是否禁用 Boolean false
readonly 设置Input是否只读 Boolean false
placeholder 占位文本 String 请输入内容
name 用于设置Input的name String 无
maxlength 设置Input最大输入长度 Number 无
required 设置Input是否是必须的 Boolean false
autofocus 是否自动获取焦点 Boolean false
autocomplete 是否自动完成 String off
width 设置Input的width String 无
height 设置Input的高度 String 无
padding-left 设置Input的padding-left，当自定义图标过大时可设置。 String 无
padding-right 设置Input的padding-right,自定义图标过大时可设置。 String 无

# Input事件回调

事件 说明 回调参数
focus 获取焦点时触发 无
blur 失去焦点时触发 无
change Input值改变时触发 event
click 点击Input尾部图标时触发 无
clear 点击清空按钮时触发 无
search 点击搜索按钮时触发 无
togglePassword 点击查看密码按钮时触发 无
keyupEnter 释放回车键时触发 event
inhibiting 在输入字符达到最大限制长度时触发 value,maxlength

# Input插槽

插槽名 说明
head Input前置内容
footer Input后置内容

方法 说明 参数
focus 用于手动获取焦点 无
blur 用于手动失去焦点 无
