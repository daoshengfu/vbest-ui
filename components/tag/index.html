# Tag标签

对信息进行标记，突出重要信息。

# 基本使用

默认标签
dot 标签
可关闭标签

属性type用于设置标签的类型，closable设置一个标签是可关闭的。

点击查看代码 
<template>
        <div class="one">
            <b-tag>默认标签</b-tag>
            <b-tag type="dot">dot 标签</b-tag>
            <b-tag closable>可关闭标签</b-tag>
        </div>
</template>

# 配合外部组件使用

标签1
标签2
标签3

属性size用于设置标签的尺寸。

点击查看代码 
<template>
    <div class="four">
      <b-tag class="tag-item" v-for="(item,index) in tagList" :key="index" closable>{{item}}</b-tag>
      <b-input v-model="value" v-show="isShow" ref="input" @keyupEnter="handleEnter"/>

      <b-button class="tag-button" type="success" size="small" height="28px" 
      @click="handleClick" v-show="isBtn">
          + new Tag
      </b-button>
    </div>
</template>
<script>
export default {
  data() {
    return {
      tagList: ["标签1", "标签2", "标签3"],
      value: "",
      isShow: false,
      isBtn: true,
    };
  },
  methods: {
    handleClick() {
      this.isShow = true;
      this.isBtn = false;
      this.$nextTick(() => {
        this.$refs.input.focus();
      });
    },
    handleEnter() {
      let str=this.value;
      this.value="";
      this.isShow=false;
      this.isBtn=true;
      this.tagList.push(str);
    },
  },
};
</script>

# API

# Tag属性

属性 说明 类型 默认值
type 标签的类型，可选值为：border，dot String border
size 标签的尺寸，可选值为：default,medium，large String default
closable 设置标签可关闭 Boolean false
close-border 设置显示关闭按钮边框 Boolean false
color 标签的样式，内置info，success,warning,error,joker，还可以传入任何颜色值 String 无
plain 标签是否镂空 Boolean false

# Tag事件回调

事件 说明 参数
click 点击标签时触发 无
close 关闭标签时触发 无
