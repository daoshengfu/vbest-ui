# menu 导航栏

# 基本使用

  • 员工管理
  • 新增员工
  • 打卡记录
  • 业绩考核

基本使用时只需要给menu属性传入一个数组即可。

点击查看代码 
<template>
    <div class="one">
        <b-menu :menu="menu"/>
    </div>
</template>
<script>
export default {
  data() {
    return {
        menu:['员工管理','新增员工','打卡记录','业绩考核'],
    };
  },
};
</script>

# 传入一个对象数组时

  • 蓝球赛事
  • 足球赛事
  • 乒乓球赛事
  • 蓝球赛事

当你需要在导航栏中使用图标、跳转路由时，只需要传入一个对象数组，并给link属性、icon属性设置值即可。

点击查看代码 
<template>
    <div class="two">
        <b-menu :menu="menu2"/>
    </div>
</template>
<script>
export default {
  data() {
    return {
        menu2:[
            {line:'',icon:'iconfont icon-icon10',content:'蓝球赛事'},
            {line:'',icon:'iconfont icon-swticonzuqiu',content:'足球赛事'},
            {line:'',icon:'iconfont icon-pingpangqiu',content:'乒乓球赛事'},
            {line:'',icon:'iconfont icon-icon10',content:'蓝球赛事'},
        ],
    };
  },
};
</script>

切换主题

属性theme用于设置导航栏主题，默认有light、dark可供选择，也可根据提供的API自定义主题。

点击查看代码 
<template>
    <div class="four" v-if="line=='4'">
      <b-menu :menu="menu2" :theme="theme" />
      <div class="group">
        <b-menu :menu="menu2" vertical :theme="theme" />
        <b-menu :menu="menu" vertical :theme="theme" />
      </div>
      <p>切换主题</p>
      <b-radio-group v-model="theme">
        <b-radio label="light">light</b-radio>
        <b-radio label="dark">dark</b-radio>
      </b-radio-group>
    </div>
</template>
<script>
export default {
  data() {
    return {
        menu:['员工管理','新增员工','打卡记录','业绩考核'],
        menu2:[
            {line:'',icon:'iconfont icon-icon10',content:'蓝球赛事'},
            {line:'',icon:'iconfont icon-swticonzuqiu',content:'足球赛事'},
            {line:'',icon:'iconfont icon-pingpangqiu',content:'乒乓球赛事'},
            {line:'',icon:'iconfont icon-icon10',content:'蓝球赛事'},
        ],
        theme: "light",
    };
  },
};
</script>

text-color用于设置导航栏不活跃时文本颜色，active-color用于设置活跃是颜色，background-color用于设置导航栏背景颜色。

    <div class="five">
        <b-menu :menu="menu" active-color="#fc378c" text-color="#f6f6f6" background-color="pink"/>
    </div>
</template>
<script>
export default {
  data() {
    return {
        menu:['员工管理','新增员工','打卡记录','业绩考核'],
    };
  },
};
</script>

# API

# Menu属性

属性 说明 类型 默认值
menu 导航栏数据 Array 无
theme 导航栏主题，可选值：light、dark String light
vertical 设置导航栏垂直 Boolean false
item-width 设置导航项宽度 String 无
item-height 设置导航项高度 String 无
background-color 设置导航栏背景颜色 String 无
text-color 设置导航栏字体颜色 String 无
active-color 设置导航栏活跃时颜色 String 无
active-background-color 设置导航栏活跃时背景颜色 String 无
hover-color 设置导航栏hover时颜色，如果不设置默认采用active-color String 无
hover-background-color 设置导航栏hover时背景颜色 String 无
active-show-border 设置导航栏活跃时是否显示边框 Boolean true

事件 说明 参数
