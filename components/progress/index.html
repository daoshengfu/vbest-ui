# Progress进度条

用于展示业务进度，常用于音视频播放器、文件上传等。

# 基础用法

# 默认用法

上面是一个最简单的进度条，只需设置属性percent即可显示进度，属性percent是已加载的百分比，建议使用.sync修饰符。

点击查看代码 
<template>
    <div class="progress-page">
            <b-progress :percent.sync="percent1"/>
    </div>
</template>
<script>
export default {
    data(){
        return{
            percent1:10,
        }
    }
}
</script>

# 配合外部组件使用

5%

通过绑定数据修改进度。

点击查看代码 
<template>
    <div class="five">
        <div class="item">
            <b-progress :percent.sync="percent6" pro-width="8px" :stroke-color="['var(--info)','var(--joker)']"/>
            <span>{{percent6}}%</span>
        </div>
        <b-button font-size="18px" @click="add">+</b-button>
        <b-button font-size="18px" @click="reduce">-</b-button>
    </div>
</template>
<script>
export default {
    data(){
        return{
            percent6:5,
        }
    },
    methods:{
    add() {
      this.percent6 += 4;
      if (this.percent6 >= 100) this.percent6 = 0;
    },
    reduce() {
      this.percent6 -= 4;
      if (this.percent6 <= 0) this.percent6 = 100;
    },
    }
}
</script>

        </div>

# 显示进度球的进度条

60%
80%

属性show-loadbar用于设置显示进度条小球，属性loadbar-color用于设置进度条小球背景颜色。属性loadbar-item-color用于设置进度条小球圆心背景。当不设置时默认以进度条颜色为背景，当进度条渐变时，以进度条第一种渐变颜色为背景。

点击查看代码 
<template>
    <div class="seven group" v-if="line=='7'">
        <div class="item">
            <b-progress :percent.sync="percent3" pro-width="6px" :stroke-color="['var(--warning)','var(--joker)']"
            show-loadbar/>
            <span>{{percent3}}%</span>
        </div>
        <div class="item">
            <b-progress :percent.sync="percent4" pro-width="6px" loadbar-color="#ccc" loadbar-item-color="#e83c3c"
            show-loadbar/>
            <span>{{percent4}}%</span>
        </div>
    </div>
</template>
<script>
export default {
    data(){
        return{
           percent3:60,
           percent4:80
        }
    },
}
</script>

# 可拖拽、可点击的进度条

20%

属性allow-drag用于设置进度条允许拖拽，该属性只在show-loadbar属性为true情况下生效，即只有显示进度条小球才能拖拽。

点击查看代码 
<template>
    <div class="eight group" v-if="line=='8'">
        <div class="item">
            <b-progress :percent.sync="percent8" pro-width="6px" :stroke-color="['var(--warning)','var(--joker)']"
            show-loadbar allow-click allow-drag is-active/>
            <div class="percent">{{getPercent8}}%</div>
        </div>
    </div>
</template>
<script>
export default {
    data(){
        return{
           percent8:20,
        }
    },
    computed:{
        getPercent8(){
            return Math.floor(this.percent8);
        }
    }
}
</script>

# API

# Progress属性

属性 说明 类型 默认值
percent 进度条的百分比，建议使用.sync修饰符 Number 0
allow-click 用于设置是否允许通过点击进度条设置进度 Boolean false
allow-drag 设置进度条可拖拽，只在show-loadbar属性为true时有效 Booleab false
vertical 设置进度条垂直显示 Boolean false
pro-width 用于自定义进度条宽度 String 无
is-active 设置进度条是否开启进度动画 Boolean false
base-color 设置进度条背景颜色 String 无
stroke-color 设置进度条颜色，可传入一个数组 String/Array 无
loadbar-color 设置进度条小球背景颜色 String 无
loadbar-item-color 设置进度条小球圆心背景色，默认为进度条颜色 String 无
show-loadbar 设置进度条是否显示进度条小球 Boolean false
show-loadbar-item 设置进度条是否显示进度条小球圆心 Boolean true

# Progress事件回调

事件 说明 参数
click 点击进度条改变进度时触发 进度条进度
dragbegin 开始拖拽时触发，返回进度条进度 进度条进度
dragging 正在拖拽时触发 进度条进度
dragend 拖拽结束时触发
