# Progress进度条

用于展示业务进度，常用于音视频播放器、文件上传等。

# 基础用法

# 默认用法

上面是一个最简单的进度条，只需设置属性percent即可显示进度，属性percent是已加载的百分比，建议使用.sync修饰符。

点击查看代码 
<template>
    <div class="progress-page">
            <b-progress :percent.sync="percent1"/>
    </div>
</template>
<script>
export default {
    data(){
        return{
            percent1:10,
        }
    }
}
</script>

# 自定义颜色、宽度进度条

属性stroke-width用于设置进度条的宽度。属性track-color用于设置进度条颜色，可接收一个数组，值为数组类型时，进度条轨道颜色渐变。属性track-base-color用于设置进度条背景颜色。

    <div class="progress-page">
            <div class="item">
                <b-progress :percent.sync="percent1" :stroke-width="8"/>
                <span>{{percent1}}%</span>
            </div>
            <div class="item">
                <b-progress :percent.sync="percent2" :stroke-width="8" track-color="var(--success)"/>
                <span>{{percent2}}%</span>
            </div>
            <div class="item">
                <b-progress :percent.sync="percent3" :stroke-width="8" track-color="var(--warning)"/>
                <span>{{percent3}}%</span>
            </div>
            <div class="item">
                <b-progress :percent.sync="percent4" :stroke-width="8" :track-color="['var(--warning)','var(--info)']"/>
                <span>{{percent4}}%</span>
            </div>
    </div>
</template>
<script>
export default {
    data(){
        return{
            percent1:10,
            percent2:40,
            percent3:60,
            percent4:90
        }
    }
}
</script>

# 配合外部组件使用

5%

通过绑定数据修改进度。

点击查看代码 
<template>
    <div class="five">
        <div class="item">
            <b-progress :percent.sync="percent6" :stroke-width="8" :track-color="['var(--info)','var(--joker)']"/>
            <span>{{percent6}}%</span>
        </div>
        <b-button font-size="18px" @click="add">+</b-button>
        <b-button font-size="18px" @click="reduce">-</b-button>
    </div>
</template>
<script>
export default {
    data(){
        return{
            percent6:5,
        }
    },
    methods:{
    add() {
      this.percent6 += 4;
      if (this.percent6 >= 100) this.percent6 = 0;
    },
    reduce() {
      this.percent6 -= 4;
      if (this.percent6 <= 0) this.percent6 = 100;
    },
    }
}
</script>

# 可点击的进度条

30%

属性allow-click允许点击进度条设置进度，实现了通过设置进度改变绑定数据。

点击查看代码 
<template>
    <div class="six group">
        <div class="item">
            <b-progress :percent.sync="percent" :stroke-width="8" :track-color="['var(--warning)','var(--joker)']"
            is-active allow-click/>
            <span>{{percent}}%</span>
        </div>
    </div>
</template>
<script>
export default {
    data(){
        return{
           percent7:30
        }
    },
}
</script>

# 显示滑块的进度条

60%
80%

属性show-thumb用于设置显示进度条滑块，属性thumb-color用于设置进度条滑块背景颜色。属性thumb-item-color用于设置进度条滑块圆心背景。当不设置时默认以进度条颜色为背景，当进度条渐变时，以进度条第一种渐变颜色为背景。

点击查看代码 
<template>
    <div class="seven group" v-if="line=='7'">
        <div class="item">
            <b-progress :percent.sync="percent3" :stroke-width="6" :track-color="['var(--warning)','var(--joker)']"
            show-thumb/>
            <span>{{percent3}}%</span>
        </div>
        <div class="item">
            <b-progress :percent.sync="percent4" :stroke-width="6" thumb-color="#ccc" thumb-item-color="#e83c3c"
            show-thumb/>
            <span>{{percent4}}%</span>
        </div>
    </div>
</template>
<script>
export default {
    data(){
        return{
           percent3:60,
           percent4:80
        }
    },
}
</script>

# 可拖拽、可点击的进度条

20%

属性allow-drag用于设置进度条允许拖拽，该属性只在show-thumb属性为true情况下生效，即只有显示进度条滑块才能拖拽。

点击查看代码 
<template>
    <div class="eight group" v-if="line=='8'">
        <div class="item">
            <b-progress :percent.sync="percent8" :stroke-width="6" :track-color="['var(--warning)','var(--joker)']"
            show-thumb allow-click allow-drag is-active/>
            <div class="percent">{{getPercent8}}%</div>
        </div>
    </div>
</template>
<script>
export default {
    data(){
        return{
           percent8:20,
        }
    },
    computed:{
        getPercent8(){
            return Math.floor(this.percent8);
        }
    }
}
</script>

# API

# Progress属性

属性 说明 类型 默认值
percent 进度条的百分比，建议使用.sync修饰符 Number 0
allow-click 用于设置是否允许通过点击进度条设置进度 Boolean false
allow-drag 设置进度条可拖拽，只在show-thumb属性为true时有效 Booleab false
vertical 设置进度条垂直显示 Boolean false
disable-transition 禁用progress渐变动画，当你频繁设置进度条且变化幅度很小时推荐使用，例如音乐播放器。 Boolean false
is-active 设置进度条是否开启进度动画 Boolean false
stroke-width
version 0.2.5
 自定义进度条宽度 Number 无
track-base-color
version 0.2.5
 设置进度条背景颜色 String 无
track-color
version 0.2.5
 设置进度条轨道颜色，可传入一个数组 String/Array 无
thumb-color
version 0.2.5
 设置进度条滑块背景颜色 String 无
thumb-item-color
version 0.2.5
 设置进度条滑块圆心背景色，默认为进度条颜色 String 无
show-thumb
version 0.2.5
 设置进度条是否显示进度条滑块 Boolean false
show-thumb-item
version 0.2.5
 设置进度条是否显示进度条滑块圆心 Boolean true

# Progress事件回调

事件 说明 参数
click 点击进度条改变进度时触发 进度条进度
dragbegin 开始拖拽时触发，返回进度条进度 进度条进度
dragging 正在拖拽时触发 进度条进度
dragend 拖拽结束时触发
