# Avatar头像

用于以图片、图标、字符等展示用户的信息/。

# 基本使用

属性size用于设置头像框的大小，shape用于设置头像框的类型。

点击查看代码 
<template>
      <div class="item">
        <b-avatar size="30px" />
        <b-avatar />
        <b-avatar size="80px" />
        <b-avatar size="110px" />

        <b-avatar shape="square" size="110px" />
        <b-avatar shape="square" size="80px" />
        <b-avatar shape="square" />
        <b-avatar shape="square" size="30px" />
      </div>
</template>

# 展示类型

user

属性icon用于设置头像显示的图标，src用于设置图片路径，也可以直接传入字符串。

点击查看代码 
<template>
      <div class="item">
           <b-avatar class="ava-item"/>
           <b-avatar class="ava-item" src="https://portrait.gitee.com/uploads/avatars/user/2534/7602838_fudaosheng_1595768126.png!avatar100"/>
           <b-avatar class="ava-item">user</b-avatar>
      </div>
</template>

# 图片加载失败

当传入的图片无法加载时，会自动显示加载失败，并会触发error事件。

点击查看代码 
<template>
      <div class="item">
        <b-avatar class="ava-item" src="avatar.jpg"/>
      </div>
</template>

# 头像遮罩

点击更换头像
点击更换头像

属性mask用于开启头像框遮罩，mask-content用于设置遮罩内部显示的字体，也可以使用插槽mask来自定义内容、开启遮罩时点击头像框会触发clickMask事件。

点击查看代码 
<template>
    <div class="item">
      <b-avatar class="ava-item" mask mask-content="点击更换头像" size="120px" />
      <b-avatar
        class="ava-item"
        mask
        mask-content="点击更换头像"
        size="120px"
        src="https://portrait.gitee.com/uploads/avatars/user/2534/7602838_fudaosheng_1595768126.png!avatar100"
      />
      
      <b-avatar class="ava-item" mask />
      <b-avatar class="ava-item" mask>
        <img
          src="https://portrait.gitee.com/uploads/avatars/user/2534/7602838_fudaosheng_1595768126.png!avatar100"
        />
        <i slot="mask" class="iconfont icon--zuidahua"></i>
      </b-avatar>
    </div>
</template>

# API

# Avatar属性

属性 说明 类型 默认值
shape 设置头像框的形状，可选值为：circle、square String circle
size 头像框的尺寸 String 50px
icon 设置头像框显示的图标,可不设置 String 无
src 设置头像框显示的图片的路径 String 无
mask 设置是否打开头像框遮罩 Boolean false
mask-content 设置遮罩内部显示的字体 String 无
font-size 设置遮罩字体大小 String 无

# Avatar事件回调

事件 说明 参数
click 点击头像框触发 无
clickMask 开启遮罩时点击头像框触发 无
error 头像框图片加载失败时触发 无

# Avatar插槽

插槽名 说明
无 用于自定义插槽展示内容
mask 用于自定义遮罩内容
