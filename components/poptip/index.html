# Poptip 气泡提示

Poptip继承了Tooptip的所有功能，并进行了扩充，适用于更多场景。 Poptip最大的特色就是支持轻量级的Modal。

# Poptip的三种触发方式

hover Title
hover content
click Title
click content
focus Title
focus content

属性trigger控制Poptip的触发方式，支持的三种触发方式：鼠标悬停、点击、聚焦，默认是点击。

点击查看代码 
<template>
    <div>
      <b-poptip class="pop-item" trigger="hover" theme="light" placement="top">
          <b-button>hover</b-button>
          <div slot="title">hover Title</div>
          <div slot="content">hover content</div>
      </b-poptip>

       <b-poptip class="pop-item" theme="light" placement="top">
          <b-button type="primary">click</b-button>
          <div slot="title">click Title</div>
          <div slot="content">click content</div>
      </b-poptip>

       <b-poptip class="pop-item" trigger="focus" theme="light" placement="top">
          <b-input placeholder="focus show poptip" height="40px"></b-input>
          <div slot="title">focus Title</div>
          <div slot="content">focus content</div>
      </b-poptip>
    </div>
</template>

# confirm模式

您确定要删除该它吗？

属性confirm用于设置Poptip为对话框模式，需要注意的是在该模式下插槽title无效。

点击查看代码 
<template>
    <div>
      <b-poptip class="pop-item" confirm theme="light" placement="top-start" @ok="handleOk" @cancel="handleCancel">
        <b-button>删除</b-button>
        <div slot="content">您确定要删除该它吗？</div>
      </b-poptip>
    </div>
</template>
<script>
export default {
  methods: {
    handleOk() {
      this.$Toast.info("点击了确定");
    },
    handleCancel() {
      this.$Toast.info("点击了取消");
    },
  },
};
</script>

# Poptip API

# Poptip属性

属性 说明 类型 默认值
trigger 设置Poptip的触发方式，可选值为：hover、click、focus String click
confirm 设置Poptip为对话框模式，该方式下插槽title无效 Boolean false
theme 设置Poptip的主题，可选值为：dark, light String dark
placement 设置Poptip显示的位置，可选值为："top","top-start","top-end","bottom","bottom-start","bottom-end","left","left-start","left-end","right","right-start","right-end", String bottom
delay 设置延迟显示提示，单位为S Number,String 0
max-length 设置Poptip提示内容的宽度，超出宽度自动换行 String 无
always 是否一直显示提示，true时事件show、hidde不会触发 Boolean false

# Poptip事件回调

事件命 说明 回调参数
show 提示显示时触发 无
hidden 提示隐藏时触发 无
cancel 点击取消按钮触发，只在confirm模式下有效 无
ok 点击确定按钮触发，只在confirm模式下有效 无

# Poptip插槽

插槽名 说明
无 Poptip主题内容
title Poptip提示的标题,在confirm模式下无效
content Poptip提示内容
