# List 列表

常用于有固定格式数据的展示，vbest中列表自带虚拟滚动条，能解决大量数据的展示问题。

vbest对list并没有进行深度封装，只是对其进行了功能扩充：滚动、分割线、斑马线等，这也就意味着可以更自由的使用list。

            </b-list>

</template>

# 斑马线

隔行变色
  • 这是list item
  • 这是list item
  • 这是list item
  • 这是list item
这是列表尾部

属性stripe用于设置开启斑马线，属性stripe-background-color用于设置斑马线背景颜色， 值得一提 的是要使用斑马线必须要给list-item设置key。

点击查看代码 
<template>
    <div class="two">
      <b-list
        show-border
        split
        stripe
        stripe-background-color="#f6f6f6"
        header="隔行变色"
        footer="这是列表尾部"
      >
        <b-list-item v-for="(item,index) in list1" :key="index">这是list item</b-list-item>
      </b-list>
    </div>
</template>
<script>
export default {
  data() {
    return {
      list1: 4,
    };
  },
};
</script>

# List API

# List属性

属性 说明 类型 默认值
header 列表表头 String 无
footer 列表表尾 String 无
item-layout 列表子元素的布局方式，可选值：horizontal,vertical String vertical
stripe 是否启用别表斑马线 Boolean false
stripe-background-color 设置斑马线背景颜色 String 无
show-border 是否显示列表边框 Boolean false
border-color 设置列表边框颜色 String 无
split 是否显示列表分割线 Boolean false
split-color 设置分割线颜色 String 无

# List插槽

插槽名 说明
header 自定义表头
无 列表主体部分
footer 自定义表尾

# ListTitle API

# ListTitle属性

属性 说明 类型 默认值
content ListTitle内容 String
split-color 分割线颜色，优先级比List高 String 无

# ListTitle插槽

插槽名 说明
无 自定义ListTitle

# ListItem API

# ListItem属性

属性 说明 类型 默认值
icon 设置ListItem的图标，和src最好只设置一个 String 无
src 设置ListItem图片路径 String 无
title 设置ListItem标题 String 无
content 设置ListItem的内容 String 无
color 设置ListItem字体颜色 String 无
split-color 设置分割线背景颜色，优先级比List高 String 无
background-color 设置ListItem背景颜色 String 无
stripe-background-color 设置斑马线背景颜色，优先级比List高 String 无

# ListItem插槽

插槽名 说明
无 自定义ListItem
