# 示例

这是一个遮罩

点击查看详情 
<template>
    <div class="one">
      <div class="item">
        <b-mask icon="iconfont icon--zuidahua" />
      </div>
      <div class="item">
        <b-mask text="这是一个遮罩" />
      </div>
      <div class="item">
        <b-mask :src="require('../public/img/icon.jpg')" />
      </div>
    </div>
</template>
<style scoped>

  height: 100px;

</style>

# Mask API

# Mask属性

属性 说明 类型 默认值
fix 遮罩是否是相对于屏幕 Boolean false
icon 设置遮罩的字体图标 String 无
src 设置遮罩要显示图片的路径 String 无
text 遮罩的字体内容 String 无
color 遮罩字体颜色 String 无
size 遮罩内字体图标、图片的大小 String 无
fontSize 遮罩字体大小 String 无
background 遮罩的背景颜色 String 无

# Mask插槽

插槽名 说明
无 自定义遮罩内容
