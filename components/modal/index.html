# modal对话框

对话框，常用于与用户交互。在VbestUI中Model有两种使用方式：使用默认封装的对话框，自定义对话框。

# 默认对话框

默认用法： 通过属性 show 来控制对话框显示/隐藏，通过属性 title 来设置对话框标题， content 设置对话框内容。属性 show-mask 是否打开遮罩。

代码示例：

<b-button type="primary" round @buttonClick="buttonClick">显示默认对话框</b-button>
        <modal :show.sync="show" z-index="22"/>

    data(){
        return {
            show:false
        }
    },
    methods:{
        buttonClick(){
            console.log('--');
            this.show=true;
        }
    }

注： 在使用属性 show 时建议使用.sync修饰符，否则要在Model对话框对应的事件回调中修改当前组件data中的show值。

# 自定义对话框

你可以根据VbestUI中modal提供的属性、插槽等来设计制作自己需要的对话框。

代码示例

<modal :show.sync="show" z-index="22" title="请选择您喜欢的运动" height="300px">
          <div class="sports">
              <ul>
                  <li><input type="checkbox" name="" id="">足球</li>
                  <li><input type="checkbox" name="" id="">篮球</li>
                  <li><input type="checkbox" name="" id="">羽毛球</li>
                  <li><input type="checkbox" name="" id="">乒乓球</li>
                  <li><input type="checkbox" name="" id="">游泳</li>
              </ul>
          </div>
      </modal>

# API

# Modal属性

属性 说明 类型 默认值
show 用于控制对话框显示/隐藏 Boolean false
title 对话框的标题，如果使用了具名插槽title自定义对话框头部，则属性title无效 String "对话框标题"
content 对话框内容，如果使用了插槽自定义对话框内容，则content属性无效 String "对话框内容"
show-mask 用于设置对话框是否使用遮罩，true为使用，建议背景为浅色时使用遮罩，背景较深时关闭遮罩。 Boolean true
mask-closable 是否允许点击遮罩关闭对话框，默认不允许 Boolean false
width 自定义对话框宽度，可不设置 String 浏览器宽度的35%
height 自定义对话框高度，可不设置 String 25vh
margin-top 自定义对话框距顶部的距离，可不设置 String 50px
z-index 自定义对话框的层级，可不设置 String 8

# Modal事件回调

事件命 说明 回调参数
modalClose 点击对话框 X 关闭对话框时的回调 无
maskClick 点击遮罩关闭对话框时的回调 无
ok 点击确定按钮的回调 无
cancel 点击取消按钮的回调 无

# Modal插槽

插槽名 说明
header 自定义对话框头部
footer 自定义对话框尾部
close 自定义对话框右上角关闭按钮
无 自定义对话框内容
