# Circle进度环

基础图表组件，常用于任务百分比展示，比条形进度环更直观。

# 基础用法

20%
40%
60%
点击查看代码 
<template>
      <div class="item">
        <b-circle :percent="percent1" />
        <b-circle :percent="percent2" :stroke-color="['var(--warning)','var(--success)']" />
        <b-circle :percent="percent3" :stroke-color="['var(--error)','var(--info)']" />
      </div>
</template>
<script>
export default {
  data() {
    return {
      percent1: 20,
      percent2: 40,
      percent3: 60,
    };
  },
};
</script>

# 配合外部组件使用

80%

属性percent用于设置进度环的百分比，属性show-icon用于设置进度环进度为100%时显示成功icon，此时进度环颜色也将变成成功时的颜色。

点击查看代码 
<template>
    <div class="two">
        <b-circle class="circle" show-icon :percent="percent4" :stroke-color="['var(--error)','var(--info)']" />
        <div>
          <b-button font-size="20px" @click="add" type="primary">+</b-button>
          <b-button font-size="20px" @click="reduce">-</b-button>
        </div>
    </div>
</template>
<script>
export default {
  data() {
    return {
      percent4: 80,
    };
  },
  methods: {
    add() {
      this.percent4 += 10;
      if (this.percent4 > 100) this.percent4 = 0;
    },
    reduce() {
      this.percent4 -= 10;
      if (this.percent4 < 0) this.percent4 = 100;
    },
  },
};
</script>

# 利用内置插槽自定义进度环

999,999
我校考研上岸人数
80%

使用VbestUI进度环内置的插槽能完成一些较复杂内容的展示。

点击查看代码 
<template>
    <div class="four group">
        <b-circle :percent="percent4" :size="180" :stroke-width="15" :stroke-color="['var(--info)','var(--success)']">
            <div class="container">
                <div class="title">999,999</div>
                <div class="info">我校考研上岸人数</div>
                <div class="percent">{{percent4}}%</div>
            </div>
        </b-circle>
    </div>
</template>
<script>
export default {
  data() {
    return {
      percent4: 80,
    };
  },
};
</script>

# API

# Circle属性

属性 说明 类型 默认值
percent 进度环的百分比 Number 0
size 设置进度条的尺寸，单位像素 Number 150
stroke-width 进度环的线宽，单位像素 Number 10
stroke-color 设置进度环的颜色，可接收数组 String,Array 无
base-color 设置进度环背景颜色 String 无
dashboard 设置进度环为仪表盘样式 Boolean false
font-size 在不使用插槽时设置进度环内字体的大小 String 24px
show-icon 设置是否在进度达100%时显示成功图标 Boolean false
icon-size 设置进度环进度达100%时icon图标大小 String 无
