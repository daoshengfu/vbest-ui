# Toast全局消息提示

用于向用户信息反馈，在顶部居中显示，有info,success,warning,error四种状态。即可局部使用，也可全局使用。 VbestUI中全局提示标签为<b-toast></b-toast>

# 局部使用

局部使用时，属性content指定提示的内容。方法show用于调用全局提示。

点击查看代码 
<template>
    <div class="box">
        <b-toast content="程序员不需要女朋友" ref="toast"/>

        <b-button type="primary" @click="handleClick1">显示提示</b-button>
    </div>
</template>
<script>
export default {
    methods:{
        handleClick1(){
            this.$refs.toast.show();
        }
    }
}
</script>

# 全局使用

全局使用时，你可以在任何页面直接调用Toast提供的全局方法来显示一个消息提示。

点击查看代码 
<template>
    <div class="box">
      <b-button type="info" @click="infoClick">显示信息提示</b-button>
      <b-button type="success" @click="successClick">显示成功提示</b-button>
      <b-button type="warning" @click="warningClick">显示警告提示</b-button>
      <b-button type="error" @click="errorClick">显示错误提示</b-button>
    </div>
</template>
<script>
export default {
  methods: {
    infoClick() {
      this.$Toast.info("这是一个普通提示");
    },
    successClick() {
      this.$Toast.success("这是一个成功提示");
    },
    warningClick() {
      this.$Toast.warning("这是一个警告提示");
    },
    errorClick() {
      this.$Toast.error("这是一个错误提示");
    },
  },
};
</script>

# 自定义时长

全局使用时info、success、warning、error方法第二个参数用于设置消息显示的时间，单位为秒，可不设置。

点击查看代码 
<template>
    <div class="box">
      <b-button type="primary" @click="timeToast">显示一个10s消息提示</b-button>
    </div>
</template>
<script>
export default {
  methods: {
   timeToast(){
      this.$Toast.info("这是一个10s消息提示",10);  
    }
  },
};
</script>

# Toast API

# Toast属性

属性 说明 类型 默认值
type 设置消息提示的类型，可选值有：info、success、warning、error String info
content 用于设置消息提示的内容 String 无
duration 设置消息提示的显示时间，单位为秒 String,Number 1.5

# Toast事件

事件名 说明 参数
show 该方法将显示消息提示，并在一定时间移除消息提示 无

# 全局使用 Toast instance

  • this.$Toast.info(content,duration)
  • this.$Toast.success(content,duration)
  • this.$Toast.warning(content,duration)
  • this.$Toast.error(content,duration)

以上方法用于全局使用消息提示，参数说明如下

参数 说明 类型 默认值
content 消息提示的内容 String 无
duration 设置消息提示显示的时长，默认1.5秒，可不设置 Number 1.5
