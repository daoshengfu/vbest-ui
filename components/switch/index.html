# Switch 开关

基本组件，常用于两种状态的切换，在VbestUI中开关标签为<b-switch></b-switch>。

# 基本开关

一个最基本的开关，你也可以通过提供的属性追加丰富的功能。

值得一提的是： 在使用开关时你需要给其v-model一个Boolean类型的值，用于开关状态判断。

点击查看代码 
<template>
    <div class="box">
            <b-switch v-model="value"/>
    </div>
</template>
<script>
export default {
    data(){
        return{
            value:false
        }
    }
}
</script>

# 三种默认尺寸   
version 0.1.8：新增size 高度

属性size用于控制开关的大小，有三个值可供选择：large、default、small，能让你更好的适用不同的场景。如果这些还不能满足你的需求，vbest-ui还可以自定义Switch大小，具体请看下面。

# 自定义开关

# 自定义颜色

属性true-color用于定义开关打开时的颜色，属性false-color用于定义开关关闭时的颜色。

点击查看代码 
<b-switch class="item" v-model="value8" true-color="var(--success)" false-color="red"/>
<b-switch class="item" v-model="value9" true-color="var(--joker)"/>
<b-switch class="item" v-model="value10" false-color="var(--warning)"/>

# 自定义尺寸   
version 0.1.8

属性width，height用于自定义尺寸，当你给它们赋值时，默认size相对应的属性将会被覆盖。

点击查看代码 
<template>
    <div class="box">
      <b-switch class="item" v-model="value11" size="large" :height="36" />
      <b-switch class="item" v-model="value12" :height="28" />
      <b-switch class="item" v-model="value13" size="small" :width="46" />
      <br />
      <b-switch class="item" v-model="value14" :width="46" :height="24" />
      <b-switch class="item" v-model="value15" :width="62" :height="32" />
      <b-switch class="item" v-model="value16" :width="82" :height="36" />
      <b-switch class="item" v-model="value17" :width="100" :height="45" />
      <b-switch class="item" v-model="value18" :width="120" :height="50" />
    </div>
</template>
<script>
export default {
    data(){
        return{
            value11: false,
            value12: false,
            value13: false,
            value14: false,
            value15: false,
            value16: false,
            value17: false,
            value18: false,
        }
    }
}
</script>

# 带状态信息的开关

开启
关闭
开
关

VbestUI开关提供的插槽,能让你自由的添加合适的开关状态描述,配合size属性效果更好。 large适用于两个汉字的场景，default适用于一个汉字或三个英文字符的场景，small适用于iconfont图标的场景。

点击查看代码 
<b-switch class="item" v-model="value5" size="large">
    <span slot="open">开启</span>
    <span slot="close">关闭</span>
</b-switch>
<b-switch class="item" v-model="value6">
    <span slot="open">开</span>
    <span slot="close">关</span>
</b-switch>
<b-switch class="item" v-model="value7" size="small">
    <div slot="open" class="iconfont icon-success"></div>
    <div slot="close" class="iconfont icon-x"></div>
</b-switch>

# 禁用的开关   
version 0.1.8：优化禁用样式
点击查看代码 
<b-switch disabled :value="true" size="small"/>
<b-switch disabled :value="true"/>
<b-switch disabled :value="false" true-color="var(--success)" false-color="red"/>

属性disabled用于设置一个图标禁用

# API

# Switch属性

属性 说明 类型 默认值
value/v-model 传给开关绑定的值，用于开关状态切换，可用v-model替代。 Boolean false
size 设置开关的大小，可选值：large、default、small String default
width 自定义开关的宽度 Number 无
height 自定义开关的高度 Number 无
disabled 设置开关是否禁用 Boolean false
trueColor 用于自定义开关打开时的颜色 String 无
falseColor 用于自定义开关关闭时的颜色 String 无
name 用于设置开关的name，可不设置 String 无

# 开关事件回调

事件命 说明 回调参数
toggle 开关状态切换时触发 无

# 开关插槽

插槽名 说明
open 自定义开关打开时显示内容
close 自定义开关关闭时显示内容
