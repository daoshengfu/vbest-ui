# Alert 警告提示

用于页面中重要消息的提示，可手动关闭，VbestUI中标签为<b-alert></b-alert>

# 默认用法

《观沧海》
东临碣石，以观沧海。 水何澹澹，山岛竦峙。 树木丛生，百草丰茂。 秋风萧瑟，洪波涌起。 日月之行，若出其中。 星汉灿烂，若出其里。 幸甚至哉，歌以咏志

最简单都用法。

点击查看代码 
<template>
  <div class="box">
      <b-alert>
        <div slot="title">《观沧海》</div>
        <div>
          东临碣石，以观沧海。
          水何澹澹，山岛竦峙。
          树木丛生，百草丰茂。
          秋风萧瑟，洪波涌起。
          日月之行，若出其中。
          星汉灿烂，若出其里。
          幸甚至哉，歌以咏志
        </div>
      </b-alert>
    </div>
</template>

# 自定义图标的Alert

《满江红》
怒发冲冠，凭栏处、潇潇雨歇。 抬望眼，仰天长啸，壮怀激烈。 三十功名尘与土，八千里路云和月。 莫等闲，白了少年头，空悲切！ 靖康耻，犹未雪。 臣子恨，何时灭！ 驾长车，踏破贺兰山缺。 壮志饥餐胡虏肉，笑谈渴饮匈奴血。 待从头、收拾旧山河，朝天阙。

使用Alert提供的icon插槽可自定义图标。

点击查看代码 
<template>
  <div class="box">
    <b-alert class="alert" type="default" closable>
        <i slot="icon" class="iconfont icon-longxiao font-style"></i>
        <div slot="title">《满江红》</div>
        <div>
          怒发冲冠，凭栏处、潇潇雨歇。
          抬望眼，仰天长啸，壮怀激烈。
          三十功名尘与土，八千里路云和月。
          莫等闲，白了少年头，空悲切！
          靖康耻，犹未雪。
          臣子恨，何时灭！
          驾长车，踏破贺兰山缺。
          壮志饥餐胡虏肉，笑谈渴饮匈奴血。
          待从头、收拾旧山河，朝天阙。
        </div>
      </b-alert>
  </div>
</template>

# 其他Alter

仅标题---物有不平则鸣，胸有郁气当出却难舒。心态年轻才是永远年轻
仅标题---白发渔樵江渚上，惯看秋月春风。
仅内容---凡是过去，皆为序曲，
仅标题 warning
仅内容 error

根据VbestUI中Alert提供的API你可以设置随意设置Alert。。

点击查看代码 
<template>
  <div class="box">
      <b-alert class="alert" closable>
        <div slot="title">仅标题---物有不平则鸣，胸有郁气当出却难舒。心态年轻才是永远年轻</div>
      </b-alert>

      <b-alert class="alert" type="info" icon closable>
        <div slot="title">仅标题---白发渔樵江渚上，惯看秋月春风。</div>
      </b-alert>

      <b-alert class="alert" type="success" icon closable>
        <div>仅内容---凡是过去，皆为序曲，</div>
      </b-alert>

      <b-alert class="alert" type="warning" icon closable>
        <div slot="title">仅标题 warning</div>
      </b-alert>

      <b-alert class="alert" type="error" closable>
        <div>仅内容 error</div>
      </b-alert>
  </div>
</template>

# Alert API

# Alert属性

属性 说明 类型 默认值
icon 设置Alert是否使用图标，默认不使用 Boolean false
type 设置Alert的类型，可选值有:default、info、success、warning、error String default
closable 设置Alert是否是可关闭的 Boolean false

# Alert事件

事件名 说明 参数
close Alert关闭时触发 无

# Alert插槽

插槽名 说明
无 Alert的描述内容
title Alert的标题
icon 自定义Alert的图标
