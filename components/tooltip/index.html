# Tooltip文字提示

常用于鼠标经过时提示信息

# 示例

top start 提示信息
top 提示信息
top end 提示信息
left start 提示信息
left 提示信息
left end 提示信息
right start 提示信息
right 提示信息
right end 提示信息
bottom start 提示信息
bottom 提示信息
bottom end 提示信息

属性placement设置提示框出现的位置，共有12中方位可供选择。theme用于设置Tooltip的主题，有两种选择dark,light。

# 多行显示

桌上灯半黑，窗外月半明，有人觉得不够亮，有人觉得不算黑。还剩酒半壶，吐完再喝啊。

content插槽用于自定义提示内容，配合属性max-length可设计不同的样式。

点击查看代码 
<template>
      <b-tooltip>
         <b-button>这里显示的是多行文本</b-button>
         <div slot="content">桌上灯半黑，窗外月半明，有人觉得不够亮，有人觉得不算黑。还剩酒半壶，吐完再喝啊。 </div>
      </b-tooltip>
</template>

# 自定义Tooltip

辰龙，十二生肖之一，地支的第五位。在时间上表示上午7时至上午9时。属龙的人天生强运。属龙的人品高，刚毅，热情，有强烈的向上心。聪明且有才华，但缺乏思虑的耐心，做事常常半途而废。表面冷漠，其实内心极强的仁侠骨气，处处为他人着想。

# Tooltip的两种主题

dark 适用于浅色背景
light 适用于深色背景

属性theme用于设置Tooltip提示主题。

点击查看代码 
<template>
      <b-tooltip placement="top-start" class="tool-right">
         <b-button>dark</b-button>
         <div slot="content">dark 适用于浅色背景</div>
      </b-tooltip>
      <b-tooltip placement="top-end" theme="light" class="tool-right">
         <b-button>light</b-button>
         <div slot="content">light 适用于深色背景</div>
      </b-tooltip>
</template>

# 延迟提示Tooltip

dark 延迟2S显示

属性delay用于设置延迟几秒提示

点击查看代码 
<template>
      <b-tooltip placement="top-start" delay="2">
        <b-button>延迟2s显示</b-button>
        <div slot="content">dark 延迟2S显示</div>
      </b-tooltip>
</template>

# 总是显示的Tooltip

这是总是显示的提示

属性always用于设置Tooltip总是显示提示，always为true时事件show、hidden不触发。

点击查看代码 
<template>
      <b-tooltip placement="top-start" always>
        <b-button>总是显示的</b-button>
        <div slot="content">这是总是显示的提示</div>
      </b-tooltip>
</template>

# Tooltip API

# Tooltip属性

属性 说明 类型 默认值
theme 设置Tooltip的主题，可选值为：dark, light String dark
placement 设置Tooltip显示的位置，可选值为："top","top-start","top-end","bottom","bottom-start","bottom-end","left","left-start","left-end","right","right-start","right-end", String bottom
delay 设置延迟显示提示，单位为S Number,String 0
max-length 设置Tooltip提示内容的宽度，超出宽度自动换行 String 无
always 是否一直显示提示，true时事件show、hidde不会触发 Boolean false

# Tooltip事件回调

事件命 说明 回调参数
show 提示显示时触发 无
hidden 提示隐藏时触发 无

# Tooltip插槽

插槽名 说明
无 Tooltip主题内容
content 提示内容
