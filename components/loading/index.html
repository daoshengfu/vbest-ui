# 固定Loading

loading……
固定Loading:
开
关

属性fix用于设置Loading为固定定位。

点击查看代码 
<template>
<div class="three">
      <b-loading
        fix
        icon="iconfont icon-loading2"
        icon-size="46px"
        z-index="21"
        background-color="rgba(0,0,0,.4)"
        v-show="isShow"
      />
      <div class="toggle">
        <span class="title">固定Loading:</span>
        <b-switch class="item" v-model="isShow">
          <span slot="open">开</span>
          <span slot="close">关</span>
        </b-switch>
      </div>
</div>
</template>
<style scoped>
.toggle {
  display: flex;
  align-items: center;
  position: relative;
  z-index: 32;
}
.toggle .title {
  margin-right: 10px;
}
</style>

# API

# Loading属性

属性 说明 类型 默认值
size Loading尺寸，可选值有：small、default、large String default
text Loading字体 String 无
icon 自定义图标，内部提供icon-loading,icon-loading2,icon-loading3三种Loading图标 String 无
color 设置Loading颜色 String 无
iconSize 设置字体图标大小 String 无
fontSize 设置Loading字体大小 String 无
backgroundColor 设置Loading背景颜色 String 无
fix 设置Loading为固定定位 Boolean false
zIndex 设置Loading层级 String 8
width 设置Loading宽度只在不设置icon或icon插槽时有用 String 无
height 设置Loading高度只在不设置icon或icon插槽时有用 String 无

# Loading事件回调

插槽名 说明
icon 用于自定义图标
text 用于自定义Loading文字
