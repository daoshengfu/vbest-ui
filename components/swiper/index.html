# Swiper轮播图

图片轮播展示，能节省页面空间，带来良好的页面视觉体验。

<template>
    <div class="two item">
      <b-swiper
        v-model="swiper"
        :height="245"
        dots-base-color="#8391a5"
        dots-active-color="red"
        :toggle-time="autoToggleTime"
        :circle-dots="isCircle"
        :dots="dots"
        :trigger="trigger"
        :arrow="arrow"
        :easing="easing"
      >
        <b-swiper-item class="list1">1</b-swiper-item>
        <b-swiper-item class="list2">2</b-swiper-item>
        <b-swiper-item class="list3">3</b-swiper-item>
        <b-swiper-item class="list4">4</b-swiper-item>
      </b-swiper>
    </div>
</template>
<script>
export default {
  data() {
    return {
      swiper: 1,
    };
  },
};
</script>

# API

# Swiper属性

属性 说明 类型 默认值
value 轮播图的索引，可以使用v-model替代 Number 0
width 轮播图的宽度，单位px Number 500
height 轮播图的高度，单位px Number auto
toggle-time 轮播图的切换时间,单位毫秒 Number 4000
trigger 指示器的触发方式,可选值click、hover String click
dots 指示器的位置，可选值:inside、outside、none String inside
circle-dots 设置指示器的样式为圆形 Boolean false
dots-base-color 指示器的背景颜色 String 无
dots-active-color 指示器活跃时的颜色 String 无
arrow 切换箭头的显示方式，可选值：hover、always、never String hover
easing 轮播图切换动画,可选值linear,ease,ease-in,ease-out,ease-in-out String ease

# Swiper事件回调

事件 说明 参数
click 点击轮播图时触发，返回索引值 value
change 轮播图切换时触发，返回激活前的索引，当前索引 oldValue,value
